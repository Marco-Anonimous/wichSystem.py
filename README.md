# WhichSystem - Identificación de Sistema Operativo por TTL

## Descripción
Herramienta desarrollada en Python para identificar el Sistema Operativo de un dispositivo basado en su dirección IP y su valor TTL.  

⚠️ **El dispositivo debe estar activo para obtener resultados.**

## Instalación y permisos  
Antes de ejecutar el script, es necesario otorgarle permisos de ejecución:  
```bash
chmod +x wichSystem.py
